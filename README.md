# 📱 Modern React Calculator - CodeAlpha Task 

A high-performance, responsive calculator built with **React**, **Vite**, and **Tailwind CSS**. This project features a dual-theme UI (Dark/Light mode), full keyboard support, and robust mathematical logic.

---

## 🚀 Live Demo
[Link](https://calculator-js-mu-two.vercel.app/)

## ✨ Features
- **Dual Theme:** Seamlessly toggle between Light and Dark modes.
- **Keyboard Support:** Full interaction using your physical Numpad or keyboard keys.
- **Advanced Logic:** - Handles floating-point arithmetic precisely.
  - Scientific notation for very large/small results.
  - Error handling for division by zero.
- **Responsive Design:** Mobile-first approach using Tailwind CSS.
- **Glassmorphism UI:** Modern aesthetic with smooth transitions and active scaling effects.

## 🛠️ Tech Stack
- **Framework:** [React.js](https://reactjs.org/) (Vite)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons:** Custom SVG (GitHub)
- **Deployment:** Vercel

## ⌨️ Keyboard Shortcuts
| Key | Action |
| :--- | :--- |
| `0-9` | Input Digits |
| `+`, `-`, `*`, `/` | Operators |
| `Enter` or `=` | Calculate Result |
| `Backspace` | Delete Last Digit |
| `Delete` | Clear Entry (CE) |
| `Escape` | Reset All (AC) |
